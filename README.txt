- El programa es un juego que trata de no chocar con los asteroides que
van viniendo.

- El jugador tiene la opcion de disparar para destruir a los asteroides.

- El puntaje obtenido se guarda y el usuario tiene la opcion de continuar
  la partida desde ese momento o empezar una nueva partida.

controles:

-te mueves con las teclas derecha, izquierda, arriba, abajo.

-para disparar presiona la tecla 'a'.
- Para pausar presiona INTRO.
